# Lumi

Lumi is a soft, playful journal and task space for daily planning, reflection, and wellbeing.

## Features
- Private sign up and sign in
- Preferred name greeting
- Calendar with tasks assigned by date
- Task drawer with drag-and-drop on desktop
- Daily canvas for notes
- Daily mode and hydration check-in
- Per-task goal trackers for tasks with target dates
- Mobile-friendly PWA layout

## Files
- `index.html`
- `styles.css`
- `script.js`
- `manifest.json`
- `sw.js`
- `app-config.js`
- `database.sql`
- `icons/`

## Setup
1. Create a Supabase project.
2. Run `database.sql` in Supabase SQL editor.
3. Add keys to `app-config.js`.
4. Deploy to Vercel, Netlify, or GitHub Pages.
